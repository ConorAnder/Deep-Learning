The following models are linear:

$$y=w_0 + w_1x_1 + w_2x_2 + \epsilon$$
$$y = w_0 + w_1x_1^2 + \epsilon$$
$$y = log(x_1)w_1 + log(x_2 + 1)(w_2 + 1) + \epsilon$$
$$y = w_1x_1 + e^{-0.7} \epsilon$$

The following are not:

$$y = w_1^2x_1 + \epsilon$$
$$y = w_1w_2 + x_1x_2 + \epsilon$$
$$y = log(w_1)x_1 + sin(w_2x_2) + \epsilon $$
$$y = wx + x \epsilon$$
$$y = \frac{w_1x_1}{w_2x_2} + \epsilon$$
